# text-adventure-game

A fun text-based adventure game made in Python!

Choose between two exciting paths:  
- **Jungle Adventure**  
- **Horror House**

Features:
- Interactive choices that change the story
- Sound effects for key moments (welcome, danger, win/lose)
- Multiple endings (win or game over)

## How to play
1. Clone the repo:  
   `git clone https://github.com/Ayush9863/text-adventure-game.git`
2. Install required library:  
   `pip install playsound3`
3. Put all sound files (.mp3 / .wav) in the project folder
4. Run the game:  
   `python main.py`
